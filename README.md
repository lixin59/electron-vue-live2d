# electron-vue3-live2d
此项目为用vue3、electron实现的桌面看板娘  
**本项目包含的live2d文件仅供学习使用请勿用于商业**
## 效果预览
![](https://cdn.jsdelivr.net/gh/q-mona/mona@latest/images/electron-live2d.gif)
## 运行
```
npm install // 安装依赖
npm run electron:serve // 编译项目
npm run electron:build // 打包项目
```
